# Welcome to Vey

Vey is a minimal programming language built for experimentation and clarity. Explore code, run it live, and learn quickly with our interactive playground.

Features include simple syntax, powerful constructs, and real-time execution. Perfect for learning programming concepts or rapid prototyping.

# About Vey

Vey is inspired by modern languages like Lua, Python, and Rust. Its core philosophy centers around minimalism and readability, making it perfect for scripting, learning language design, and rapid experimentation.

Key features include:

  • Simple Syntax: Clean and readable code structure
  • Dynamic Typing: Flexible variable types
  • Built-in Functions: Common operations out of the box
  • Interactive Environment: Real-time code execution
